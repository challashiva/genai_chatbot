GenAI Chatbot — Full Project Blueprint Full Stack

A multi-role AI chatbot platform with React frontend, Python/FastAPI backend, and GenAI (Groq/Claude) integration. Roles: Admin, User, Customer — each sees different features. Built phase by phase so you learn as you go.

Core Pillars
1
Frontend — React + Vite, React Router, Zustand (state), TailwindCSS
2
Backend — Python FastAPI, JWT auth, SQLite → PostgreSQL
3
GenAI Layer — Groq SDK, streaming responses, prompt templates per role
4
Auth & Roles — Login system, JWT tokens, role-based route guards
5
Admin Panel — User management, usage analytics, model config

Why this architecture?
→
Separation of concerns — each file does one job. Swap Groq for OpenAI by changing only ai_service.py.
→
Streaming first — SSE is simpler than WebSockets for one-direction AI token delivery.
→
Security at the backend — frontend hides UI, backend enforces auth on every API call.

How role enforcement works in code
# Backend: FastAPI role dependency
def require_role(required: str):
    def checker(user = Depends(get_current_user)):
        if user.role != required:
            raise HTTPException(403, "Forbidden")
        return user
    return checker

# Usage on admin-only route:
@router.get("/admin/users")
async def get_users(admin = Depends(require_role("admin"))):
    ...
// Frontend: RoleGuard component
const RoleGuard = ({ role, children }) => {
  const { user } = useAuthStore()
  if (user?.role !== role) return null
  return children
}
// Usage in JSX:
<RoleGuard role="admin">
  <AdminPanel />
</RoleGuard>
Frontend React
→
Vite — fast dev server, replaces Create React App
→
React Router v6 — page routing, nested routes, protected routes
→
Zustand — simpler than Redux, stores auth + chat state globally
→
Axios — HTTP client with interceptors for JWT header injection
Backend Python
→
FastAPI — async, auto OpenAPI docs, great for streaming + Depends injection
→
SQLAlchemy — ORM for DB models (SQLite in dev, PostgreSQL in prod)
→
python-jose — JWT creation and verification (HS256 algorithm)
→
passlib[bcrypt] — secure one-way password hashing
GenAI AI Layer
→
Groq SDK — free tier, llama-3.1-8b-instant model, fast streaming
→
System prompts per role — admin gets full access, customer gets scoped bot
→
SSE streaming — tokens flow browser-side word by word via ReadableStream
